# Moments For Mom 💗

An emotional, premium Mother’s Day web experience built with **React**, **Tailwind CSS**, and **Framer Motion**.

## Features

- Hero with animated **“Thank You Mom ❤️”** text
- Floating hearts + soft pastel aesthetic + elegant transitions
- **Photo memory gallery** with upload + lightbox preview
- Animated **“Reasons I Love My Mom”** cards (+ add your own)
- Memory **timeline** (+ add your own)
- **Voice message**: record (MediaRecorder) or upload audio + player
- AI‑inspired **poem generator** (runs in-browser)
- **Digital greeting card generator** with confetti + download/share
- Soft background **ambient music** (WebAudio) with mute toggle
- Dark / light mode toggle
- Fully responsive modern UI
- “Made with TRAE” footer

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
